
# Lotto Number Generator

## Overview

This is a simple web application that generates and displays a set of 6 random lottery numbers. The application is built using modern HTML, CSS, and JavaScript, with a focus on a clean, responsive, and engaging user experience.

## Features & Design

*   **UI Components:**
    *   A main container with a title.
    *   A display area for the generated numbers, styled to look like lottery balls.
    *   A "Generate Numbers" button with an interactive "glow" effect.
    *   An animated "Lotto Ball" graphic for visual appeal.
*   **Styling:**
    *   **Colors:** A vibrant color palette with a main theme color and accent colors. CSS variables are used for easy theming.
    *   **Typography:** Expressive fonts to create a clear visual hierarchy.
    *   **Layout:** A centered, responsive layout that works on both desktop and mobile devices.
    *   **Effects:**
        *   Subtle noise texture on the background for a premium feel.
        *   Multi-layered drop shadows on containers to create depth.
        *   A "glow" effect on the interactive button.
*   **JavaScript Logic:**
    *   Generates 6 unique random numbers between 1 and 45.
    *   Displays the numbers dynamically in the UI.
    *   Handles the button click event.
